# ArcGIS API for JavaScript 开发示例

## 项目介绍

基于ArcGIS API for JavaScript 开发的简单GIS系统。

### 技术选型

* jQuery

* Bootstrap

* ArcGIS API for JavaScript

* Bootstrap Duallistbox-双列表框插件

* baiduTemplate-Javascript模板引擎

### 开发工具

* SVN: 版本管理

* Webstorm: 开发IDE
